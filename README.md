# AIgame
## 概述：
基本功能：人为指定游戏框架，ai生成剧情和选项，玩家选择选项，并根据选项继续生成接下来的剧情
像这样https://game.ur1.fun/adarkroom
扩展选项：
1多模态，ai生成音效 ai生成带语调的声音 ai生成场景图片（类似galgame）
2 游戏数值 加入游戏数值以及通关条件等以增加可玩性，如血条 好感度等。
## 开发流程：
1. 世界观设定
1.单个npc语言风格设计+1v1对话
2.上帝设计（可能的下一步剧情、选项）
3.控制者设计（只生成命令，作为独立agent）
4.集成（群聊对话形式）
5.前端
6.多模态
7.外部工具
8.文档及安全性检查
